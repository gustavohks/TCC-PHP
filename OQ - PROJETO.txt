OQ - PROJETO
Sistema web que possibilita ao usuário a criação de formulários de forma rápida e dinâmica, com edição dos componentes. Ferramenta de geração do código fonte, além de geração do sistema com integração do formulário com banco de dados. 

PQ - AJUDAR EMPRESAS QUE PRECISAM DE TAL POR MEIO DE TAL
Facilitar o processo de criação de formulários, ou seja, otimizar o tempo de desenvolvimento de sistemas de cadastro tanto para programadores inexperientes como para empresas que desejam um processo mais ágil de desenvolvimento.
